# SupportLink (고객 지원 포털)

**SupportLink**는 고객 문의 데이터를 안전하게 관리하고 프론트엔드와 데이터를 교환하기 위한 안정적인 RESTful API 백엔드 시스템입니다.

## 문서 (Documentation)
* **최신 버전:** [Backend API v1.0](./docs/Backend_API_v1.0.md)

## 🛠 기술 스택 (Tech Stack)
* **API:** RESTful API (JSON)
* **인증:** JWT (Access + Refresh Token)
* **데이터베이스:** RDBMS (MySQL 권장, PostgreSQL 등)
* **스토리지:** AWS S3 / MinIO (파일 첨부용)
* **권장 언어/프레임워크:** Java (Spring Boot) 권장 또는 Node.js (Express/NestJS), Python (Django/FastAPI)

## 주요 기능 (Key Features)
1. **인증 (Authentication):** 회원가입, 로그인, 토큰 관리
2. **FAQ / 지식 기반:** FAQ 조회, 검색 및 관리 (관리자)
3. **티켓 관리 (Tickets):** 티켓 생성, 조회, 상태 변경
4. **티켓 답변 (Replies):** 티켓에 대한 답변 작성 및 소통
5. **(v1.1) 파일 첨부:** 티켓 및 답변에 이미지/문서 첨부 기능
6. **(v1.1) 내부 협업:** 상담원 전용 내부 메모(Internal Notes) 기능
7. **(v1.1) 알림:** 티켓 상태 변경 및 답변 등록 시 이메일 알림
